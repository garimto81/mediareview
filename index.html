유튜브 영상 피드백,
타임코드로 소통하는 비결

더 이상 "아까 그 부분이요..." 라고 말하지 마세요. 타임코드를 사용하면 영상의 정확한 시점을 콕 집어, 누구나 명확하게 이해하는 피드백을 남길 수 있습니다.

1. 직접 입력 (기본)

가장 기본적이면서 확실한 방법입니다. PC와 모바일 어디서든 사용할 수 있습니다.

입력 형식:

MM:SS (분:초) HH:MM:SS (시:분:초)

사용 예시:

02:45 화면 전환이 너무 빠른 것 같아요.
15:10 배경음악 소리가 갑자기 커져요.

댓글을 등록하면 시간이 파란색 링크로 자동 변환되어 클릭 시 해당 시간으로 바로 이동합니다.

2. PC에서 가장 편리한 방법: URL 복사

클릭 두 번으로 가장 정확하고 빠르게 타임코드를 만들 수 있습니다. 다른 메신저로 공유할 때도 매우 유용합니다.

  1. 피드백을 남길 장면에서 영상을 **일시 정지**합니다.
  2. 영상 화면에 **마우스 오른쪽 버튼**을 클릭합니다.
  3. 나타나는 메뉴에서 **"현재 시간에 동영상 URL 복사"**를 선택합니다.
  4. 댓글 창에 붙여넣기(`Ctrl+V`)하면 완료!

복사되는 링크 형식:

https://youtu.be/영상고유주소?t=165

시간이 초 단위로 정확하게 입력되어 오차가 없습니다.

PRO-TIP: 체계적인 피드백 댓글 작성법

여러 피드백을 남길 때는 한 댓글에 목록처럼 정리해 보세요. 피드백을 확인하고 수정하는 사람이 훨씬 편하게 작업할 수 있습니다.
